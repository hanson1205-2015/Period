Features

Why write in Period?

Period blends the precision of programming with the flow of natural language.

📖

Readable by Design

Statements read like sentences. Keywords such as let, be, and show make intent obvious.

🎯

Pinpoint Diagnostics

Parse and semantic errors report the exact line and column range. Multiple issues are reported in a single pass.

⚡

Native JIT

Programs compile to native code with Cranelift by default. Numeric loops also get closed-form and periodic optimisations that eliminate whole loops at compile time.

📦

Modules & Standard Library

Import built-in modules such as math, string, random, and time. Local modules and relative imports are supported too.

🧠

Turing Complete

Variables, conditionals, loops, recursive functions, lists, dictionaries, and classes provide full computing power.

Performance

Benchmark overview.

Nine workloads covering numeric loops, strings, lists, function calls, classes, and exception handling. Lower is better.

Sum 1..N

Benchmarked with benchmark_long.py (averaged over 10 runs). Workloads: summing 1..N, counting numbers divisible by 3 or 5, counting numbers divisible by 3 and 5, summing numbers divisible by 3 or 5, string concatenation, list growth, function calls, object instantiation, and try/catch. C is compiled with the best available optimizing compiler (MSVC, GCC, Clang) and falls back to the bundled TCC if none are installed; Rust, Go, C# and Java use their normal release builds. Period now compiles all programs through its Cranelift-based JIT by default.

Syntax

Reads like a sentence.

-- Declare a variable.
let name be "Period".
show name.

Keywords like let, be, and show make intent obvious at a glance.

Diagnostics

Pinpoint error messages.

hello.period:2:11: error: Expected '.' at end of statement.
     2 | show name
       |           ^

The parser recovers from errors and reports every issue with exact line and column ranges.

Tooling

code --install-extension period-language.vsix
Functions

Recursive and readable.

-- A recursive factorial.
define factorial with n:
    if n <= 1 then:
        return 1.
    return n * factorial with (n - 1).

Define functions with plain English and call them with with.

Examples

See it in action.

Click a tab to switch examples. The same sentence-like style runs everywhere.

-- Greet the world.
let greeting be "Hello, World!".
show greeting.
-- Count to five.
let i be 1.
while i <= 5 repeat:
    show i.
    set i to i + 1.
-- A recursive factorial.
define factorial with n:
    if n <= 1 then:
        return 1.
    return n * factorial with (n - 1).
-- A tiny class.
class Person:
    init with name:
        set this name to name.
    define greet with greeting:
        show greeting + ", " + this name + "!".

let ada be new Person with "Ada".
tell ada to greet with "Hello".
