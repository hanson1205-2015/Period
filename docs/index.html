Period.

An English programming language designed to read like an article. Every statement ends with a period, giving your code a calm, literary rhythm.

Get Started Read the Docs

Why Period?

Readable by Design

Statements read like sentences. Keywords such as let, be, show, and define make intent obvious.

Detailed Diagnostics

Errors point to the exact line and column. The parser recovers from mistakes so you see every issue in one pass.

Turing Complete

Variables, conditionals, loops, and recursive functions give Period the full power of a modern language.

Tooling Included

A VS Code extension with syntax highlighting, hover help, autocompletion, formatting, and go-to-definition.
