Features

Why write in Period?

Period blends the precision of programming with the flow of natural language. It is built for developers who believe code should be as pleasant to read as it is to write.

📖

Readable by Design

Statements read like sentences. Keywords such as let, be, show, and define make intent obvious at a glance.

🎯

Pinpoint Diagnostics

Errors report the exact line and column range. The parser recovers from mistakes so you see every issue in one pass.

⚡

Fast Compiler

The interpreter is implemented in Python and packaged into a single native executable with Nuitka for quick compile and run cycles.

🧠

Turing Complete

Variables, conditionals, loops, recursive functions, lists, and dictionaries provide the full power of a modern language.

🛠️

VS Code Extension

Syntax highlighting, hover help, autocompletion, formatting, go-to-definition, and LSP diagnostics out of the box.

🔁

Interactive REPL

Run period without arguments to start a REPL. Type statements ending with a period and see results instantly.

How it works

Write. Run. Read.

Three simple steps from an idea to a running program.

1

Write your program

Create a .period file and write statements that end with a period. Add comments with -- to explain your logic.

2

Run with one command

Use period file.period to compile and execute. The compiler reports precise errors if anything is wrong.

3

Enjoy readable output

Your program runs and prints results. The code itself remains clean, article-like, and easy to maintain.
