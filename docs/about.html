About Period

Period is an educational programming language designed to explore what happens when code is forced to look like prose. By requiring every statement to end with a period, Period encourages authors to write programs that feel deliberate and complete.

Goals

  • Readable, sentence-like syntax
  • Helpful, precise error messages
  • Modern tooling: LSP, formatting, and IDE integration
  • Full Turing completeness

The Name

Every statement in Period ends with a period. That single character is both a delimiter and a philosophy: finish what you start. It turns a source file from a pile of instructions into a paragraph of intent.

Who It Is For

  • Students meeting their first programming language
  • Teachers who want syntax that reads out loud in class
  • Developers curious about language design and interpreters
  • Anyone who believes code should be pleasant to read

Technology Stack

  • Language runtime: Python 3 with a hand-written lexer, parser, and tree-walking interpreter
  • Executable packaging: Nuitka compiles the runtime into a single Windows executable
  • LSP server: JSON-RPC language server served over stdio from the same executable
  • VS Code extension: TypeScript client built on vscode-languageclient
  • Documentation site: Static HTML/CSS with no build step

Roadmap

  • Richer standard library (file I/O, math, dates)
  • Package manager
  • Bytecode compiler and faster VM
  • Cross-platform executables (macOS and Linux)
  • Online playground in the browser

Get Involved

Period is open source and welcomes contributions. Report bugs, suggest syntax ideas, or submit pull requests on the GitHub repository. Discussion and design notes live in the repository issues.
