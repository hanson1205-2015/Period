Installation

Windows Installer

Download the latest installer from the GitHub Releases page. It will:

  • Install period.exe to C:\Program Files\Period
  • Add Period to your system PATH
  • Install the VS Code extension

Run a Program

period hello.period

Start the REPL

period

Type statements ending with a period. Use exit. or Ctrl+C to quit.

VS Code Extension

If the installer did not install the extension, run:

code --install-extension period-language.vsix
