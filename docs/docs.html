Language Documentation

Every statement in Period ends with a period (.). Comments begin with two dashes and extend to the end of the line.

Basics

A Period program is a sequence of statements. Each statement ends with a period. Whitespace and blank lines are ignored.

-- This is a comment.
let x be 10.
show x.

Variables

Use let to declare a variable and set to reassign an existing variable or index.

let name be "Period".
set name to "Period Language".

let items be [1, 2, 3].
set items[0] to 10.

Data Types

Numbers

Integers and floating-point numbers are supported.

let count be 42.
let pi be 3.14.

Strings

Strings are enclosed in double quotes and support \n, \t, \", and \\ escapes.

let message be "Hello, Period.".

Booleans and Nothing

let flag be true.
let empty be nothing.

Lists

Lists are ordered collections. Use brackets for indexing.

let numbers be [1, 2, 3].
show numbers[0].
set numbers[0] to 10.

Dictionaries

Dictionaries store key-value pairs. Keys can be strings, numbers, or booleans.

let person be {"name": "Ada", "age": 36}.
show person["name"].
set person["age"] to 37.

Operators

Arithmetic

  • + addition (also string/list concatenation)
  • - subtraction and unary negation
  • * multiplication (also string repetition)
  • / division
  • % modulo
  • ** power

Comparison

==, !=, <, >, <=, >=

Logical

and, or, not

Conditionals

let score be 85.
if score >= 60 then.
    show "pass".
otherwise.
    show "fail".
end if.

Loops

Use while with repeat to loop while a condition is true.

let i be 0.
while i < 10 repeat.
    show i.
    set i to i + 1.
end while.

Functions

Define functions with define and call them with with. Use return to send a value back.

define greet with name.
    show "Hello, " + name + "!".
end define.

greet with "World".

define factorial with n.
    if n <= 1 then.
        return 1.
    end if.
    return n * factorial with n - 1.
end define.

Built-in Functions

  • length with x — length of a string, list, or dictionary
  • string with x — convert a value to a string
  • number with x — convert a value to a number
  • type with x — return the type name of a value
  • input — read a line from standard input
show "What is your name?".
let name be input.
show "Nice to meet you, " + name + ".".

Error Messages

Period reports errors with the exact line and column range. The parser recovers from errors so multiple problems can be reported in a single pass.

example.period:1:11: error: Expected '.' at the end of a 'let' statement.
       1 | let x be 5
         |           ^
