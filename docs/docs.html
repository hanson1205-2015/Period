Language Documentation

Basics

Every statement in Period ends with a period (.). Comments begin with two dashes.

-- This is a comment.
let x be 10.
show x.

Variables

Use let to declare and set to reassign.

let name be "Period".
set name to "Period Language".

Conditionals

let score be 85.
if score >= 60 then.
    show "pass".
otherwise.
    show "fail".
end if.

Loops

let i be 0.
while i < 10 repeat.
    show i.
    set i to i + 1.
end while.

Functions

define greet with name.
    show "Hello, " + name + "!".
end define.

greet with "World".

Built-ins

  • length with x — length of string, list, or dictionary
  • string with x — convert to string
  • number with x — convert to number
  • type with x — type name
  • input — read a line of input

Data Structures

let items be [1, 2, 3].
let person be {"name": "Ada", "age": 36}.
show items[0].
show person["name"].
